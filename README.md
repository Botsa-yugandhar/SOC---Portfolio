# SOC Analyst Portfolio

This repository contains Security Operations Center (SOC) investigation projects.
The goal of this portfolio is to demonstrate practical blue team investigation skills including log analysis, threat detection, and MITRE ATT&CK mapping.

## Projects

1. Windows Brute Force Attack Investigation
2. Suspicious PowerShell Execution Analysis
3. Phishing Email Malware Investigation
4. Network Traffic Investigation (Wireshark)

## Tools Used

- Splunk
- Wireshark
- Nmap
- VirusTotal
- Windows Event Logs
- Linux Authentication Logs

## Skills Demonstrated

- Log analysis
- Incident investigation
- Threat detection
- MITRE ATT&CK mapping
- Detection rule creation
